Character builder

Future plan for character builder. (Main focus of the project)

  • Players can select a weapon for specific character. A datasheet will show calulated stats scaling for each level.
  • Players can select a weapon for specific character. A datasheet will show calulated stats scaling for each level.
  • Players can select individual artifacts/artifact sets to build a character.
  • Players can select artifact stats and substats manually to create prefered artifact to be used in character building.
  • Stats scaling for level and ascension.
  • Potential damage graph.
  • Reaction calculation and damage graph scaling with each type of reaction.
  • Show farming route and date in calendar view. (Character ascension, talent ascension and selected weapon ascension.)
  • (Maybe) Show common and local material routes.
